# Basic UI Kit
Basic UI Kit es una librería de componentes de React que proporciona un conjunto básico de componentes de interfaz de usuario para ayudar a construir aplicaciones web más rápida y fácilmente.

## Instalación
Para instalar Basic UI Kit, simplemente ejecute el siguiente comando en su proyecto de React:

```bash
npm install basic-ui-kit
```

```bash
yarn add basic-ui-kit
```

## Uso
Para usar Basic UI Kit en su proyecto, importe los componentes que necesite en su archivo de componente de React:

```react
import { Button, Form, Input } from 'basic-ui-kit';
```
Luego, puede usar los componentes importados en su componente de React:

```react
function MyComponent() {
  return (
    <div>
      <Button>Click me</Button>
      <Form>
        <Input placeholder="Enter your name" />
      </Form>
    </div>
  );
}
```
